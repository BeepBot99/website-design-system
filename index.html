Libraries

Prism.js

Prism.js is a library for formatting code snippets. It is better than its competitors for a few reasons. First of all, it recognizes syntax a lot better, especially with newer syntax. Second of all, it uses smaller files and has quicker loading times. Furthermore, Prism.js makes it very easy to customize the styles, unlike some other formatting libraries. Lastly, it has loads of plugins, developed by both the people who made it and the community, and you can even make your own plugin if you want. The code snippets on this website were even formatted with Prism.js!

Normalize.css

Different browsers have different default css styles. One thing you can do to change this is by adding a CSS reset. A CSS reset can be as simple as body {margin: 0; padding: 0}, or it can be really advanced. Normalize.css is a CSS reset that makes every default CSS property the same on every browser, but it doesn't change anything that is consistent across browsers. That way, you don't have to worry about your website looking different on another browser, especially Internet Explorer. In addition, unlike other libraries, it is very easy to read and understand the code, and is very well commented.

Hover.css

Have you ever seen a cool hover effect but didn't know how to replicate it? Or have you wanted to put really cool hover effects on your buttons? If, so you should check out Hover.css. They have over 50 hover effects for buttons, and all you have to do to add one is put a class on your button! Although it is primarily used for buttons, you can put it on any element that you want to apply a hover affect to. When I first found out about this, I was so amazed because I had seen all those hover effects before, and I could finally put them on my websites!

Animate.css

When I first found out about Animate.css, I thought, "This is useless. You can already do all of this with CSS animations." Then, I started looking at the demos. You can use one CSS property for basically any animation you want to make! It can make it slide in and out, bounce around, and even turn to jello! Moreover, you can control the timing just like you can with the built-in CSS animations. If you ever need any CSS animations, simple or complicated, I would recommend using Animate.css.
