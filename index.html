    box-sizing: border-box;
}

    margin: 0;
    font-size: 16px;
}

Center Child Element(s)

.container { display: flex; justify-content: center; /* Horizontally */ align-items: center; /* Vertically */ flex-direction: column; /* Only for multiple child elements stacked vertically */ }

Default Font Sizes

h1 { font-size: 2em; } h2 { font-size: 1.5em; } h3 { font-size: 1.3em; } h4 { font-size: 1em; } h5 { font-size: 0.8em; } h6 { font-size: 0.7em; }

Transition Timing Functions

.div { transition-timing-function: ease; /* Default value, starts slow, fast in the middle, ends slow */ transition-timing-function: ease-in-out; /* Like ease but more smooth */ transition-timing-function: ease-in; /* Starts slow, then fast, then abrupt stop at the end */ transition-timing-function: ease-out; /* Fast, abrupt start, slows down to a stop at the end */ transition-timing-function: linear; /* Same speed the entire transition */ }

.container { overflow: hidden; } .container img { max-width: 100%; height: auto; display: block; }

Grid Properties

grid-template-rows and grid-template-columns

Specifies the width and number of rows/columns. For example, grid-template-columns: 20% 40px 60%;.

grid-template

Shorthand property. Syntax is grid-template: grid-template-rows / grid-template-columns;.

fr unit

Divides the remaining space. For example, grid-template-rows: 1fr 20px 3fr;. The middle row gets 20px, the first row gets 1/4 of the remaining space, and the third column gets 3/4 of the remaining space.

row-gap and column-gap

Sets the gaps in the rows\columns. For example, row-gap: 10px;.

gap

Shorthand for row-gap and column-gap. Syntax is gap: row-gap column-gap;. Alternatively, you can set both at once. For example, gap: 20px;. This sets the gap for both the rows and columns at the same time.

repeat() function

Repeats a value a certain number of times. For example, grid-template-columns: repeat(6, 50px);. This makes 6 columns, all with width 50px. You can also repeat multiple values. For example. grid-template-rows: repeat(2, 20px 50px);. This is the same as saying grid-template-rows: 20px 50px 20px 50px;.

minmax() function

Fills all remaining space without going below the minimum or above the maximum. Syntax is minmax(minimum, maximum). For example, grid-template-columns: 10px minmax(20px, 30px) 10px;. This will make the first and third columns each 10px wide, and the middle column will fill remaining space as possible, but will never go under 20px or above 30px.

grid-row-start and grid-column-start

Unlike the other properties, this is put on a grid item instead of the grid container. It specifies what row/column the item should start on. For example, grid-row-start: 2;. This means that the topmost row of the grid item will be the 2nd row.

grid-row-end and grid-column-end

Is equal to one more than the last row/column for a grid item. For example, consider grid-column-end: 4;. In this example, the grid item will end on the column just before the 4th column, so it will end on the 3rd column.

span keyword

Can be put on either grid-row-start or grid-row-end, but only one. It can also be put on either grid-column-start or grid-column end. It specifies how many rows/columns a grid item will span. For example, grid-row-start: 3; grid-row-end: span 4;. The grid item will start on row 3 and take up 4 rows total. So, it will take up rows 3, 4, 5 and 6. This is the same as saying grid-row-start: 3; grid-row-end: 7;.

grid-row and grid-column

Shorthand properties for grid-row: grid-row-start / grid-row-end; and grid-column: grid-column-start / grid-column-end;.

grid-area

Shorthand for grid-area: grid-row-start / grid-column-start / grid-row-end / grid-column-end;.

justify-items and align-items

Specifies how grid items will align themselves in their grid spaces. justify-items controls left to right, or the inline axis, and align-items controles up and down, or the block axis. Values are stretch, which is the default, start, which aligns to the top or left, end, which aligns to the bottom or right, and center, which aligns to the center.

justify-self and align-self

Gives you the ability to control justify-items and align-items for a specific grid item. For example, justify-self: end;.

justify-content and align-content

Only applies if the grid container is bigger than the grid itself. Controls where in the grid container the grid will go. Values are all of the values for justify-items and align-items, in addition to three more. space-around puts an equal amount of space around each row/column, resulting in an equal amount of space on each end and double that amount in between rows/columns. space-between puts an equal amount of space in between rows/columns, with no space on either end. Lastly, space-evenly puts the same amount of space on ends and in between rows/columns.

grid-auto-flow

Specifies where to put grid items if there isn't any space for them in the grid. This is called an implicit grid. Values are row, which fills unfilled rows and then creates new rows, column, which fills unfilled columns and then creates new columns, and dense, which attempts to fill any holes earlier in the grid if smaller elements are added. You can also combine row or column with dense. For example, grid-auto-flow: column dense;.

grid-auto-rows and grid-auto-columns

Specifies the width of rows and columns added by grid-auto-flow. For example, grid-auto-columns: 50px 20%;. The first column added by grid-auto-flow will be 50px wide, the second will be 20%, the third will be 50px, and so on.
