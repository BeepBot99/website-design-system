Styles

Use Everywhere

* {
    box-sizing: border-box;
}
body {
    margin: 0;
    font-size: 16px;
}

Center Child Element(s)

.container {
    display: flex;
    justify-content: center; /* Horizontally */
    align-items: center; /* Vertically */
    flex-direction: column; /* Only for multiple child elements stacked vertically */
}
